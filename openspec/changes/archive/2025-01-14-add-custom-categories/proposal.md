# Change: Add Custom Categories

## Why
Users need the ability to create custom transaction categories to better organize their financial tracking beyond default categories.

## What Changes
- Add categories management screen
- Implement create/edit/delete custom categories
- Add icon and color picker for category customization
- Update category picker to show custom categories
- Add "Manage Categories" link in settings
- **BREAKING**: None (backward compatible)

## Impact
- Affected specs: categories
- Affected code: app/categories/, components/Category*, database/operations/categories.ts
